#  VyomGarud — Next-Gen UAV Systems Landing Page

## ⚙️ Overview
**VyomGarud** is a modern, cinematic landing page for a next-generation **UAV / Drone Systems** company.  
It represents **precision, reliability, and advanced autonomy** — built with **React.js** and **Tailwind CSS**.  
This single-page website captures the professional and futuristic spirit of drone innovation.

---



##  Tech Stack
-  **React.js** — Component-based frontend framework  
-  **Tailwind CSS** — Utility-first styling for modern UI  
-  **Framer Motion** — Subtle animations and smooth transitions  
-  **Vite** — Super-fast development and build tool  
-  **GitHub Pages** — Easy and free hosting  

---

##  Features
- Transparent **Navbar** with smooth scroll navigation  
- Cinematic **Hero Section** with a drone video background and strong CTA  
- Elegant **About Section** with transparent glassmorphic cards  
- **Capabilities Section** with creative layout and optional video background  
- Stylish **Highlights Section** with animated feature points  
- Modern **Contact Form** with gradient hover effects  
- Interactive **Footer** with click-based background color transition
